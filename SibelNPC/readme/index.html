What does this mod contain?

Sibel NPC is a content mod for Improved Anvil. It introduces a new character companion â€“ Sibel Arcanis, a young swashbuckler who enlists the protagonist's help in uncovering the truth behind the conspiracy that had caused the demise of her ancestors. The mod features:

  1. Sibel, a playable companion character who may join the party early on. She's a Swashbuckler thief with options to be dual-classed into Cleric or Fighter.
  2. Two large quests that span almost the entire length of the Bhaalspawn saga, filled with many tactical challenges.
  3. A new NPC, the artificer lady Elowen, who offers an assortment of fresh item upgrades, featuring both brand new items and improvements to the existing recipes.
  4. Many small patches, improvements and additions to the original Improved Anvil mod.

Installation

Sibel NPC is compatible with Improved Anvil: EE 7.0 and the classic Improved Anvil 6.0 both. The mod ships with the modern version of WeiDU by default. If you're installing Sibel NPC on a classic ToB installation, make sure to replace the default version of WeiDU with WeiDU v185 (the same executable you used to install IA 6.0).

The installation procedure is fairly straightforward. Just unpack the mod into your BG2 folder and run the WeiDU installer. Make sure to install all components that are offered for your particular version of the game. If you don't have Improved Anvil installed already, the mod will fail, for obvious reasons. If you're updating to a newer version of the mod, make sure that you uninstall the previous one first and completely replace the mod's folder before proceeding.

If you are planning to install any components of IATweaks, do so after you install Sibel NPC.
